srtb-visualizer demo

Basic usage

Code

const visualizer = new SRTBVisualizer(srtbJson, {
    difficulty: RatingDifficulty.EASY,
});

let container = document.querySelector("#container");
container.appendChild(visualizer.getCanvas());
visualizer.render();

Demo

With controls

Code

const visualizer = new SRTBVisualizer(srtbJson, {
    difficulty: RatingDifficulty.EASY,
    controls: true,
});

let container = document.querySelector("#container");
container.appendChild(visualizer.getCanvas());
visualizer.render();
